## ⚡ Quick Start (5 Minuten)

### 1. GitHub Repository erstellen
- Gehe zu [github.com/new](https://github.com/new)
- Name: `vachsystems`
- Public ✓
- Create repository

### 2. Pushen
```bash
cd vach-systems

# Ersetze YOUR-USERNAME mit deinem GitHub-Benutzernamen!
git remote add origin https://github.com/YOUR-USERNAME/vachsystems.git
git push -u origin main
```

### 3. GitHub Pages aktivieren
- Settings → Pages
- Source: `main` branch, `/root` folder
- Save

### 4. Fertig!
```
https://YOUR-USERNAME.github.io/vachsystems/
```
