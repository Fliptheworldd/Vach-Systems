Open Graph Image Generator

Erstellt ein 1200x630px Bild für WhatsApp/Social Media Vorschauen

Anleitung:

  1. Klick auf "OG-Image herunterladen"
  2. Speichere die Datei als og-image.jpg
  3. Lade sie in den vach-systems/ Hauptordner hoch
  4. Fertig! WhatsApp zeigt jetzt das Logo in der Link-Vorschau

Tipp: Du kannst das Design im Code anpassen (Zeile 50-80)
